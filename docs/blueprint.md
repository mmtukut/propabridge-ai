# **App Name**: Propabridge AI

## Core Features:

- AI-Powered Property Search: Find properties using AI to understand natural language search queries, filtering by location, budget, and preferences, connecting to Whatsapp in the backend as a tool for results delivery
- Verified Listings: Ensure trust and safety with a verification process for all listed properties, including document verification, physical inspections, and owner identity confirmation.
- Fraud Prevention: Implement fraud detection and prevention measures to protect users from fake listings and scams.
- AI-Powered Listing Optimization: Provide AI-driven recommendations and optimizations for property listings to improve visibility and attract potential renters/buyers; serves as a tool for optimizing copy in listing.
- Direct Landlord Contact: Enable direct communication between property seekers and landlords, fostering transparency and streamlined transactions.
- Personalized User Dashboard: Create a personalized dashboard for renters and landlords to track their searches, listings, and communications.
- Admin Management Dashboard: Enable administrators to approve or deny property listings to enhance listing verification procedures

## Style Guidelines:

- Primary color: Warm Trust Blue (#2B5CE6). Evokes reliability and tech-savviness.
- Accent color: Vibrant Coral (#FF6B6B). Brings energy and Airbnb-style warmth.
- Background color: Light gray (#F7F7F7). Provides a clean, modern backdrop.
- Body font: 'Inter', a grotesque-style sans-serif known for its modern, objective, and neutral appearance, which can be applied across headlines or body text. Note: currently only Google Fonts are supported.
- Headline font: 'Space Grotesk', a proportional sans-serif with a computerized, techy, scientific feel.
- Use Lucide React icons, maintaining consistency and clarity throughout the interface.
- Airbnb-inspired layout with rounded corners, generous white space, and trust-building elements to create a warm and inviting user experience.
- Subtle animations and micro-interactions to enhance user engagement and provide visual feedback.
